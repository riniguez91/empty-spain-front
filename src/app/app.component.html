¿Qué es el Análisis de Sentimiento?

El Análisis de Sentimiento o minería de opinión es un tipo de procesamiento del lenguaje natural para realizar el seguimiento del estado de ánimo del público sobre un producto en particular. El análisis de sentimiento implica la construcción de un sistema para recoger y categorizar opiniones sobre un producto. Conocido también como minería de opinión, puede automatizarse usando machine learning, un tipo de inteligencia artificial (IA), para extraer el sentimiento del texto

Esta aplicación consiste en analizar un texto introducido en el campo dedicado y se obtienen como resultado unos valores de positividad y negatividad para este texto.
